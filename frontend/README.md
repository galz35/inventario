# Sistema de Inventario - Frontend

Desarrollado con React + Vite + TypeScript.

## Requisitos
- Node.js v18+
- npm v9+

## Instalación
```bash
npm install
```

## Ejecución (Desarrollo)
```bash
npm run dev
```

## Estilo
Se utiliza **Vanilla CSS** con variables globales en `src/styles/global.css` para mantener un diseño premium, rápido y sin dependencias externas pesadas.
Los colores corporativos son:
- Primario: Rojo (#d32f2f)
- Secundario: Verde (#2e7d32)
- Fondo: Negro/Antracita (#121212)

## Pantalllas Principales
1. **Panel Principal (Dashboard)**: Métricas rápidas de valor de stock y tareas del día.
2. **Estado de Almacenes**: Listado de consumibles por bodega y técnico.
3. **Logística de Traslados**: Control de transferencias "En Tránsito".
4. **Reportes y Auditoría**: Consumo por proyecto y ajustes de inventario.
5. **Ejecución de OT (Técnico)**: Flujo paso a paso con firma y evidencias.
6. **Activos Serializados**: Control de equipos únicos por número de serie.
